## How to run:
 ### Using Vercel:
  just go to this URL and use it.
 
 ### Running this project from local:
  * Download this repo:
  ```git clone git@github.com:kiqecanijo/todo-dgraph.git```
  * update dependencies:
  ```npm i``` or ```yarn```
  * Run it:
  ```yarn start```
